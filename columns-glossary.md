# Описание колонок мастер-таблицы settlements-climate-master.xlsx

Excel содержит **5 листов**. Ниже — построчные пояснения к каждой колонке каждого листа.

---

## Лист 01_master_settlements (главная таблица, 205 строк)

| Колонка | Тип | Описание |
|---|---|---|
| `id` | строка | Машинный идентификатор (на латинице, snake_case) — например `moscow`, `chelyabinsk`. Используется в JSON и в коде. |
| `country` | строка | Страна. Сейчас везде «Россия». |
| `region` | строка | Субъект РФ (область / край / республика / автономный округ). |
| `settlement` | строка | Название населённого пункта на русском. |
| `settlement_type` | строка | Тип: «город», «пгт», «село», «деревня». |
| `snow_region` | римская цифра | Снеговой район по карте 1 СП 20.13330.2016 (I…VIII). |
| `sg_kpa` | число (кПа) | Нормативное значение веса снегового покрова Sg, кПа. Округлено до табличного значения района (см. 02_value_maps). |
| `snow_source` | строка | Источник Sg: ссылка на СП и/или сверочный калькулятор. |
| `snow_status` | enum | Статус данных: `verified` / `from_sp20_appendix_k` / `requires_verification` / `manual` / `not_applicable`. |
| `wind_region` | римская цифра | Ветровой район по карте 3 СП 20.13330.2016 (Ia, I…VII). |
| `w0_kpa` | число (кПа) | Нормативное значение ветрового давления w0, кПа. |
| `wind_source` | строка | Источник w0. |
| `wind_status` | enum | Статус ветровых данных. |
| `ice_region` | римская цифра | Гололёдный район по карте 4 СП 20.13330.2016 (I…V). |
| `ice_thickness_mm` | число (мм) | Толщина стенки гололёда b на высоте 10 м, мм. |
| `ice_source` | строка | Источник гололёдных данных. |
| `ice_status` | enum | Статус гололёдных данных. |
| `seismic_points` | число (баллы) | Сейсмичность по картам ОСР-2015, карта А (массовое строительство). Карты B и C хранятся в outputs/ese-data.json (для НВФ обычно достаточно А). |
| `seismic_source` | строка | Источник сейсмики. |
| `seismic_status` | enum | Статус сейсмических данных. |
| `terrain_type_default` | enum | Тип местности по умолчанию (СП 20, табл. 11.2). Сейчас везде null — инженер должен задать вручную. |
| `terrain_type_allowed` | строка | Допустимые типы местности через запятую — `A, B, C`. |
| `manual_allowed` | bool | Можно ли инженеру переопределить тип местности вручную. |
| `expert_override_allowed` | bool | Можно ли при экспертизе переопределить значения нагрузок. |
| `data_status` | enum | Общий статус записи: `verified` / `partial` / `requires_verification`. |
| `comment` | строка | Свободный комментарий: контрольный город или нет, какие значения уточнены через ese.pro, особые примечания. |

## Лист 02_value_maps (справочники)

| Колонка | Описание |
|---|---|
| `Тип справочника` | Категория: «Снеговой район (Sg)», «Ветровой район (w0)», «Тип местности», «Гололёдный район», «Сейсмика». |
| `Код` | Кодовое значение: римская цифра района / буква типа местности. |
| `Значение` | Числовое значение (если применимо). |
| `Единицы` | кПа / мм / м/с² / баллы. |
| `Описание` | Что означает: например «Sg = 1.0 кПа — для III снегового района». |

Используется как «расшифровщик» — по коду района из 01_master_settlements можно вытащить численное значение из этого листа.

## Лист 03_control_checks (15 контрольных городов)

Подмножество 01_master_settlements: контрольные точки для ручной сверки.

| Колонка | Описание |
|---|---|
| `settlement` | Название города. |
| `id` | id записи (для cross-reference). |
| `snow_region` | Снеговой район. |
| `sg_kpa` | Sg, кПа. |
| `wind_region` | Ветровой район. |
| `w0_kpa` | w0, кПа. |
| `ice_region` | Гололёдный район. |
| `ice_thickness_mm` | b, мм. |
| `seismic_points` | Сейсмика, баллы. |
| `data_status` | Должно быть `verified`. |
| `comment` | Свободный комментарий. |

15 городов: Москва, СПб, Челябинск, Екатеринбург, Казань, Новосибирск, Краснодар, Пермь, Уфа, Самара, Ростов-на-Дону, Тюмень, Омск, Владивосток, Якутск.

## Связь между листами

```
01_master_settlements        ←—— основная запись по городу
   ↓ id
03_control_checks           ←—— подмножество для ручной сверки (15 строк)
   ↓ id
05_climate_extras           ←—— расширенная схема (СП 22 + СП 131)

02_value_maps               ←—— справочник кодов→значений (район I→Sg=0.5)
04_sources_plan             ←—— список использованных нормативов и сайтов
```

## Статусы (enum) — что значат

| Статус | Что это |
|---|---|
| `verified` | Значение сверено через ese.pro и совпадает с СП. Можно использовать. |
| `from_sp20_appendix_k` | Из приложения К СП 20.13330.2016 (печатное издание / официальные таблицы). |
| `requires_verification` | Сейчас стоит null или предположение — нужно подтверждение по СП. |
| `manual` | Введено вручную инженером, без автоматической сверки. |
| `not_applicable` | Не применяется (например, сейсмика 5 баллов для НВФ практически неактивна). |
